1.2 RDP（Remote Desktop Protocol）

  • 适用场景：远程控制 Windows 系统桌面。
  • 主要功能：图形化远程控制，适合需要操作 GUI 的任务。

操作步骤：

  1. 确保服务器已启用远程桌面（Windows 系统中设置）。
  2. 使用客户端工具：
    • Windows：使用自带的 远程桌面连接（mstsc）。
    • macOS 和 Linux：安装 Microsoft Remote Desktop 或 Remmina。

4. 提示与建议

  1. 启用防火墙：配置 UFW 或 iptables 保护服务器。

  2. 使用强密码：确保所有账户都使用强密码，避免被暴力破解。

  3. 定期更新系统：保持服务器和软件更新以减少漏洞风险。

  4. 启用密钥认证：对于 SSH，推荐使用密钥登录替代密码登录。
