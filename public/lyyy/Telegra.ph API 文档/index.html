Telegra.ph API 文档

以下是 Telegra.ph API 文档的全参版本，按照你要求的格式进行书写：

1. createAccount

请求方法

POST https://api.telegra.ph/createAccount

请求参数

参数名 类型 是否必填 描述 示例值
short_name string 必填 帐户的简短名称，1-32 个字符。 “XiaoYang”
author_name string 可选 作者名字，显示在页面的作者信息中，0-128 个字符。 “Xiao Yang”
author_url string 可选 作者的个人页面或社交媒体链接，0-512 个字符。 “https://lyone.xyz“

响应示例

2. revokeAccessToken

请求方法

POST https://api.telegra.ph/revokeAccessToken

请求参数

参数名 类型 是否必填 描述 示例值
access_token string 必填 当前帐户的访问令牌。 “d3b25feccb89e508a9114afb82aa421fe2a9712b963b387cc5ad71e58722”

响应示例

3. editAccountInfo

请求方法

POST https://api.telegra.ph/editAccountInfo

请求参数

参数名 类型 是否必填 描述 示例值
access_token string 必填 帐户的访问令牌。 “d3b25feccb89e508a9114afb82aa421fe2a9712b963b387cc5ad71e58722”
short_name string 可选 更新后的简短名称，1-32 个字符。 “XiaoYangNew”
author_name string 可选 更新后的作者名字，0-128 个字符。 “New Xiao Yang”
author_url string 可选 更新后的作者的个人页面链接，0-512 个字符。 “https://new-lyone.xyz“

响应示例

4. getAccountInfo

请求方法

POST https://api.telegra.ph/getAccountInfo

请求参数

参数名 类型 是否必填 描述 示例值
access_token string 必填 帐户的访问令牌。 “d3b25feccb89e508a9114afb82aa421fe2a9712b963b387cc5ad71e58722”
fields string 可选 返回字段列表，以 JSON 数组格式传递。 ["short_name","page_count"]

响应示例

5. createPage

请求方法

POST https://api.telegra.ph/createPage

请求参数

参数名 类型 是否必填 描述 示例值
access_token string 必填 帐户的访问令牌。 “d3b25feccb89e508a9114afb82aa421fe2a9712b963b387cc5ad71e58722”
title string 必填 文章标题，1-256 个字符。 “我的第一篇文章”
content string 必填 文章内容，格式为 JSON 编码的节点对象数组。 [{ "tag": "p", "children": ["Hello world!"] }]
author_name string 可选 覆盖帐户默认的作者名字。 “Xiao Yang”
author_url string 可选 覆盖帐户默认的作者链接。 “https://lyone.xyz“
return_content bool 可选 如果为 true，返回文章的内容。 true

响应示例

6. editPage

请求方法

POST https://api.telegra.ph/editPage

请求参数

参数名 类型 是否必填 描述 示例值
access_token string 必填 帐户的访问令牌。 “d3b25feccb89e508a9114afb82aa421fe2a9712b963b387cc5ad71e58722”
path string 必填 页面路径。 “My-First-Article-09-05”
title string 必填 更新后的文章标题，1-256 个字符。 “我的修改后的文章”
content string 必填 更新后的文章内容，格式为 JSON 编码的节点对象数组。 [{ "tag": "p", "children": ["Updated content."] }]
author_name string 可选 覆盖帐户默认的作者名字。 “New Xiao Yang”
author_url string 可选 覆盖帐户默认的作者链接。 “https://new-lyone.xyz“
return_content bool 可选 如果为 true，返回更新后的文章内容。 true

响应示例

7. getPageList

请求方法

POST https://api.telegra.ph/getPageList

请求参数

参数名 类型 是否必填 描述 示例值

offset int 可选 从第几个页面开始返回（分页）。 0
limit int 可选 返回的页面数量（分页）。 10
