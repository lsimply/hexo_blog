常见徽章平台和类型

在文档或项目页面中添加徽章是一种很常见的做法，尤其在 GitHub 仓库或文档站点中，用于展示项目状态、构建状态、版本号、许可证等信息。

常见徽章平台和类型

  1. Shields.io：提供免费的徽章生成服务，支持各种样式的徽章。
  2. Travis CI、CircleCI、GitHub Actions 等 CI 工具：提供构建状态徽章。
  3. npm、PyPI 等包管理器：提供项目版本、下载量等徽章。
